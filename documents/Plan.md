# План автоматизации тестирования веб-сервиса «Путешествие дня»

## 1. Перечень автоматизируемых сценариев

### 1.1. Методы генерации тестовых данных
- Эквивалентное разбиение
- Анализ граничных значений

### 1.2. Позитивные сценарии

**Сценарий 1: Успешная оплата по дебетовой карте**

Входные данные:
- Номер карты: 1111 2222 3333 4444
- Месяц: 12
- Год: 26
- Владелец: Ivan Petrov
- CVC: 123

Ожидаемый результат: сообщение «Успешно. Операция одобрена банком»
Проверка БД: в таблице payment_entity статус APPROVED

**Сценарий 2: Успешная оплата в кредит**

Входные данные:
- Номер карты: 1111 2222 3333 4444
- Месяц: 08
- Год: 27
- Владелец: Maria Ivanova
- CVC: 987

Ожидаемый результат: сообщение «Успешно. Операция одобрена банком»
Проверка БД: в таблице credit_request_entity статус APPROVED

### 1.3. Негативные сценарии

**Сценарий 3: Дебетовая оплата отклонённой картой**

Входные данные:
- Номер карты: 5555 6666 7777 8888
- Месяц: 12
- Год: 26
- Владелец: Ivan Petrov
- CVC: 123

Ожидаемый результат: сообщение «Ошибка! Банк отказал в проведении операции»
Проверка БД: статус DECLINED

**Сценарий 4: Кредит отклонённой картой**

Входные данные: аналогично сценарию 3, форма «Купить в кредит»
Ожидаемый результат: сообщение «Ошибка! Банк отказал в проведении операции»

**Сценарий 5: Пустые поля**

Поочерёдное оставление пустым каждого поля формы. Остальные поля заполнены валидными данными.

Ожидаемый результат: сообщение «Неверный формат» или «Поле обязательно»

Проверяемые поля: номер карты, месяц, год, владелец, CVC.

**Сценарий 6: Невалидный формат полей**

Номер карты:
- 15 цифр
- 17 цифр
- 16 нулей
- буквы
- карта не из набора (1234 5678 9012 3456)

Месяц:
- 00
- 13
- одна цифра
- буквы

Год:
- истёкший год (20)
- 00
- 105
- буквы

Владелец:
- кириллица
- цифры в имени
- одна буква
- спецсимволы

CVC:
- одна цифра
- две цифры
- четыре цифры
- буквы

Ожидаемый результат для всех перечисленных случаев: сообщение «Неверный формат» или «Истёк срок действия карты» для истёкшего года.
