# Отчёт по итогам автоматизации

## 1. Что было запланировано
- Разработать автотест для позитивного сценария (оплата одобренной картой)
- Автотест должен проверять:
  - Заполнение формы
  - Отправку данных
  - Появление сообщения «Успешно»
  - Сохранение статуса `APPROVED` в базе данных

## 2. Что реализовано
- Автотест `test_payment_positive.py` написан и запускается
- Использованы: Python, pytest, Selenium, psycopg2
- Проверка базы данных (PostgreSQL) добавлена
- Вывод автотеста сохраняется в консоли

## 3. Результаты
- Тест успешно выполняет все шаги до проверки базы данных
- **Найден баг:** после успешной оплаты данные не сохраняются в таблице `payment_entity`
